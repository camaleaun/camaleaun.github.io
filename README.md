# camaleaun.github.io

This repository is the publishing target for the blog static output.

## Contribution policy

Please do not open feature Pull Requests directly against this repository.

If you want to contribute content or code changes:

1. Fork the source repository: https://github.com/camaleaun/blog
2. Create your branch in your fork of camaleaun/blog.
3. Open your Pull Request to camaleaun/blog.

## Why

This repository is updated by deployment automation from camaleaun/blog and can be overwritten by new publishes.

## Source of truth

- Source repository: https://github.com/camaleaun/blog
- Published site target: https://github.com/camaleaun/camaleaun.github.io
